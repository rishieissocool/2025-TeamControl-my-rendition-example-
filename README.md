# Team Control 

This is the Repository for Our main computer control.

This repository contains : 

1. [Network](/src/TeamControl/Network) -> Communications

2. [World](/src/TeamControl/World) -> Anything Related to the simulation world

3. [Roles](/src/TeamControl/Roles) -> Roles of the Robot and Behaviour

4. [Strategy](/src/TeamControl/Strategy) -> Decision Making and Team strategy 

5. [Shared](https://github.com/WSU-TurtleRabbit/shared.git) -> Shared module with RobotClient 

--- 

## Installation 

To install this module, do

```
git clone https://github.com/WSU-TurtleRabbit/team-control.git
```

Then activate the install.sh file and install

- LINUX
```
chmod 755 install.sh
./install.sh
```
*if you have encountered a permission issue, please see install.sh file.*

- WINDOWS
Open New Terminal : Git bash
```
./install.sh
```
