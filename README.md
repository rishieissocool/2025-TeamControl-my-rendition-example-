# Team Control (2025)

This is the Repository for Our server-side operation

---

## Installation 

To install this module, do

```
git clone https://github.com/WSU-TurtleRabbit/2025-teamcontrol.git
```

Then activate the install.sh file and install

```shell
./install.sh
```

For linux (Ubuntu), and Windows User, you can use virtual environment in the script
